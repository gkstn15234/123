# 오토데일리 (AutoDaily)

자동차 경제 전문 미디어 사이트

## 🚗 소개

오토데일리는 자동차 산업과 경제 정보를 전문적으로 다루는 뉴스 사이트입니다.

### 주요 특징
- 경향신문 스타일의 모던한 디자인
- 구글 디스커버 최적화
- 반응형 웹 디자인
- SEO 최적화

### 카테고리
- 🚗 **자동차**: 신차 정보, 자동차 산업 뉴스
- 💰 **경제**: 자동차 관련 경제 분석, 시장 동향

## 🔧 기술 스택

- **Static Site Generator**: Hugo
- **CSS Framework**: Bootstrap 5
- **Icons**: Font Awesome
- **Hosting**: GitHub Pages / Cloudflare Pages
